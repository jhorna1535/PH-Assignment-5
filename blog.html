Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

How do you selected an element form the DOM?

Use an ID to Select an Element with .getElementById() Use a CSS Selector to Select an Element with .querySelector() Use a CSS Selector to Select Multiple Elements with .querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a design pattern in JavaScript used to efficiently manage and handle events on multiple child elements by attaching a single event listener to a common ancestor element.

How do you manipulate an element`s attributes and styles using the DOM?
  • 1. Manipulating Attributes: The setAttribute, getAttribute, and removeAttribute methods allow you to manipulate an element's attributes (such as id, href, src, etc.).
  • 2. Manipulating Styles: You can modify an element's styles either by directly manipulating its style property or by using classList to add or remove CSS classes.
